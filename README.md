# Core Node 2.5 | Prestige HQ 🛠️
**Core Node 2.5** — это модульная среда разработки и виртуальный хаб для управления спецификациями миров, разработанный в рамках экосистемы *Prestige HQ*.
### 🚀 Основные возможности
 * **Context Keeper:** Интегрированный блокнот с поддержкой Markdown-форматирования для ведения заметок по проекту. Все данные автоматически сохраняются в локальном кэше браузера.
 * **Экспорт данных:** Мульти-форматный экспорт спецификаций (.worldpack, .json, .txt) для легкой интеграции с внешними системами.
 * **Oracle Chat:** Адаптивное ядро взаимодействия с потоками сессий для оперативного решения задач.
 * **Mobile-First UX:** Адаптивная архитектура, позволяющая работать как на десктопе, так и на мобильных устройствах с помощью системы оверлеев.
### 🛠 Технологический стек
 * **Frontend:** Чистый HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript.
 * **UX/UI:** Material Symbols Rounded, Material You цветовая палитра.
 * **Storage:** LocalStorage для управления состоянием сессии и сохранения контекста.
### 📦 Быстрый старт
 1. Разверните файл index.html на хостинге (или встройте через iframe в Google Sites).
 2. Используйте Context Keeper для записи метаданных текущего проекта.
 3. Используйте терминал для команд взаимодействия с ядром.
 4. Выгружайте готовые конфиги через блок **Экспорт Спецификации**.
